## ðŸ†˜ Need Help?

**Common issues:**

**"Wallet not connecting"**
- Make sure MetaMask/Coinbase Wallet installed
- Check if Base network is added
- Try refreshing page

**"Transaction failed"**
- User needs ETH on Base for gas (~$0.01)
- Make sure they're on Base network, not Ethereum mainnet
- Check they have enough USDC

**"How do I verify payment?"**
- Go to https://basescan.org
- Paste transaction hash
- Verify "To" address is yours: 0xB4ac8Da2C07a43f52CA04080348e1afadDe2A651
- Verify amount matches tier price
