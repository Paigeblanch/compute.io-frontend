## üêõ Common Issues & Solutions

**"Wallet won't connect"**
- Install MetaMask or Coinbase Wallet
- Enable browser extension
- Check if popup was blocked

**"Wrong network"**
- Code automatically switches to Base
- If manual: Chain ID is 8453
- RPC: https://mainnet.base.org

**"Transaction failed"**
- Check wallet has enough ETH for gas (~$0.01)
- Check wallet has USDC balance
- Verify contract address is correct

**"Can't find pages"**
- Check routing is set up
- Verify file paths are correct
- Check imports in App.tsx
